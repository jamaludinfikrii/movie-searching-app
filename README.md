# Movie Searcing App (IMDB API)


### 1. Clone this repository

```bash
$ git clone https://github.com/jamaludinfikrii/movie-searching-app.git

$ cd movie-searching-app
```

### 2. Install all dependency needed
```bash
$ npm install
```

### 3. Run Application
```bash
$ npm start
```
